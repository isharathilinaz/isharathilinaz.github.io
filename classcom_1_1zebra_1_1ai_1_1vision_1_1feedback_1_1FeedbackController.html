Detailed Description

Manages and triggers different types of feedback (audio, haptic, LED) in the AI vision system.

The FeedbackController class provides methods to configure and trigger audio feedback using predefined sounds or custom URIs, enable or disable haptic feedback, and set LED feedback colors. It supports feedback for events like barcode scans.

void com.zebra.ai.vision.feedback.FeedbackController.trigger ( )
inline

Triggers all enabled feedback mechanisms.

This method checks which feedback types are configured and triggers them accordingly:

  • If an audio feedback is set (either through an enum or URI), it plays the sound.
  • If haptic feedback is enabled, it activates vibration.
  • If an LED color is set, it lights up the LED in the specified color.

Feedback is triggered in the order: audio, haptic, then LED. This method should be called when a feedback event (e.g., barcode scan success) occurs.
