Detailed Description

Interface representing a generic trackable entity within the AI vision system.

The Entity interface defines the methods required for any trackable entity, including retrieving bounding box data, metadata, user data, and performing equality checks.

Template Parameters
TThe type of entity that extends this interface, ensuring type safety in comparisons.
